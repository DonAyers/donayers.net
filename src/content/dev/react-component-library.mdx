## Overview

A modern, accessible component library designed for rapid development of web applications. Features dark mode support, full TypeScript coverage, and comprehensive documentation.

### Key Features

- 🎨 **30+ Components** - Buttons, forms, modals, cards, and more
- ♿ **Accessible** - WCAG 2.1 AA compliant
- 🌗 **Dark Mode** - Built-in theme switching
- 📱 **Responsive** - Mobile-first design approach
- 🎭 **Customizable** - Easy theming with Tailwind
- 📚 **Documented** - Storybook integration

### Tech Stack

- **React 18** with hooks and modern patterns
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Storybook** for documentation
- **Vitest** for testing
- **Rollup** for bundling

### Usage Example

```tsx
import { Button, Card } from '@donayers/components';

function App() {
  return (
    <Card variant="cyberpunk">
      <h2>Welcome</h2>
      <Button variant="neon" onClick={() => console.log('Clicked!')}>
        Click Me
      </Button>
    </Card>
  );
}
```
