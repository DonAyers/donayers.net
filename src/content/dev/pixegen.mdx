## Overview

pixegen is a small toolset for generating pixel art assets programmatically. The goal is to make it easy to produce consistent sprite sheets, tilesets, or character variations without having to hand-draw each frame — useful in game dev workflows where you want a lot of visual variety without a lot of manual art effort.

*Experimental — early stage.*
