## Overview

Battlestation is a VS Code extension built around a simple frustration: local scripts and automation tasks accumulate across projects, and there's no good place to see or run them without digging through terminals and READMEs.

The extension adds a centralized dashboard panel inside VS Code where you can organize scripts — npm tasks, shell commands, build steps, whatever — and execute them in one click. The goal is a tighter dev loop: everything in one place, no context switching.

### What it does

- Surfaces local scripts (npm, shell, custom) in a sidebar panel
- Run any task directly from the UI without touching a terminal
- Organize and label tasks across a project or workspace
- Built in TypeScript against the VS Code extension API
